FAQs

The base URL is
http://apisandbox.openbankproject.com
Please make sure you are using this in all your API calls

  1. Avoid using trailing slashes, else, you would get a 404 error. Example:

    .../obp/v1.4.0 200 OK
    .../obp/v1.4.0/ 404 Not Found
  2. Double check parameters are spelt correctly (including http vs https etc.)
  3. Check your encoding (use UTF8)

There are multiple ways to authenticate a user including: OAuth and Direct Login . If you are using this sandbox for a hackathon, we recommend you use Direct Login to start with.

If you want to use OBP with OAuth, we recommend you use (and fork) one of our OAuth Starter SDKs. If you are using this sandbox for a hackathon, we recommend you use Direct Login. For an OAuth walkthrough example with sample code, please see here. We support OAuth 1.0a. For deepish technical details of the flow see here.
We support OAuth 2.0. For the technical details of using OBP API with OAuth 2.0, please see here.

Please use the API Explorer

You will need to login to the API as a sandbox customer. You can do this using the API Explorer, or any REST client. You can find some example credentials here.
